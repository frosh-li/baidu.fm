##百度音乐随心听命令行版本

###Installation

$ sudo npm install baidu.fm -g


###On Debian/Ubuntu, the ALSA backend is selected by default, so be sure to have the alsa.h header file in place:

$ sudo apt-get install libasound2-dev


###Run
baidu.fm

###获取git获取后
npm install
